================================================================================
                    WEBSITE DESIGN GUIDELINES
          For Creating Professional Business Websites
================================================================================

PROJECT PURPOSE
===============

This project aims to create professional websites for businesses that do not 
yet have their own online presence.


CORE PRINCIPLE
==============

Do not use emojis. They create the impression of being quickly generated by AI. 
This is exactly what we want to avoid.


WHAT TO AVOID
=============

These visual elements and patterns immediately signal AI-generated design:


VISUAL CLICHÉS
--------------

• Rounded cards with shadows and small icons in the corner
• "Hero" sections with large headings on the left and stock images on the right


TYPOGRAPHY
----------

• Overuse of Inter, Roboto, or Space Grotesk
• All text at the same size and weight with no real hierarchy
• Headings that are only slightly larger than body text


COPY & CONTENT
--------------

• "We are a passionate team that..." - this sentence appears on thousands of 
  websites
• Buzzword-filled sections using vague terms like "innovative," "customer-focused," 
  "holistic solutions"
• Three-column layouts where each box has an icon, heading, and two sentences - 
  the exact same structure every time


ICONS & IMAGES
--------------

• Direct use of Feather Icons or Heroicons with no unique visual language
• All icons exactly 24x24px, same stroke width, gray color
• Overused Unsplash stock photos that appear on countless other sites


LAYOUT STRUCTURE
----------------

• Predictable sequence: Hero → Features (3 columns) → Testimonials → CTA → Footer
• Each section exactly 100vh tall - this pattern is so common it's immediately 
  recognizable


COLOR PALETTE
-------------

• 2-3 colors where one is always blue or purple
• White as secondary and dark gray as tertiary
• Nothing surprising or industry-specific


WHAT TO DO INSTEAD
==================

TYPOGRAPHY
----------

Use less common font combinations that feel intentional. For example: 
DM Serif Display + Instrument Sans, or fonts that directly relate to the industry.


COLOR
-----

Derive colors from the actual company brand identity, not from default design 
templates.


LAYOUT
------

• Break away from the grid: use asymmetrical layouts
• Overlay text on images
• Use varying margins and spacing


CONTENT
-------

Write copy that sounds like a real person, not a marketing bot. Be authentic and 
conversational.


THE ULTIMATE TEST
=================

If the website could be any company's website without changing the logo, 
it was built using AI defaults.


SEARCH ENGINE OPTIMIZATION (SEO)
================================

A beautiful website means nothing if people cannot find it. These are essential:

• Use relevant keywords in headings and body text
• Write descriptive meta descriptions for each page
• Create a sitemap (sitemap.xml)
• Use an SSL certificate (HTTPS) - this is non-negotiable

Google will only find and rank the business if the website is properly optimized 
for search.
